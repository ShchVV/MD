Автобиография

Родилась в г.Минске 09.07.1994. С 2001 по 2012 училась в СШ №184 г.Минска. В 2010-2012 принимала участие в олимпиаде среди школьников по химии. В 2012 поступила на биологический факультет БГУ, который закончила с отличием в 2017 году по специальности "биохимия". На сегодняшний день - студент магистратуры биологического факультета БГУ по специальности "биология". Тема дипломной работы: Оптимизация методов пробоподготовки для панорамного протеомного анализа.
